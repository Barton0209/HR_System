# HR System v2.0 — Система управления персоналом

## 🚀 Быстрый старт

### Запуск на локальной машине (без root прав)

```bash
cd /workspace/hr_app
./start_server.sh
```

**Адреса доступа:**
- Локально: http://localhost:8000
- Из сети: http://<ВАШ_IP>:8000

## 📁 Структура проекта

```
hr_app/
├── backend/              # Серверная часть (FastAPI)
│   ├── routers/          # API endpoints
│   ├── services/         # Бизнес-логика
│   ├── main.py           # Точка входа
│   └── database.py       # Работа с SQLite
├── frontend/             # Клиентская часть
│   ├── index.html        # Главный HTML
│   └── static/
│       ├── css/style.css
│       └── js/*.js       # Модули JS
├── data/                 # Данные приложения
│   ├── uploads/          # Загруженные файлы
│   ├── reports/          # Сгенерированные отчеты
│   └── hr_system.db      # База данных SQLite
├── OLD/                  # Архив старых файлов
├── start_server.sh       # Скрипт запуска
└── requirements.txt      # Зависимости Python
```

## 📊 Загрузка данных (вкладка "Настройки")

### Основные файлы:
1. **БАЗА.xlsx** — основная база сотрудников (лист "База_1С")
2. **ПАРОЛЬ_ДОСТУП.xlsx** — доступы пользователей (лист "ПАРОЛЬ_ДОСТУП")
   - Колонки: Логин, Пароль, ДОСТУП, ФИО, Email, Должность, Отдел, Доступ к Карнет
3. **Подразделение_Отдел_Участок.xlsx** — справочник подразделений
4. **Терр_ПЛОЩ_ПОДР_СтатусОП_Регион.xlsx** — справочник территорий
5. **Должность, Классификация.xlsx** — справочник должностей
6. **МАРШРУТ.xlsx** — маршруты для билетов

### Реестры билетов:
- 1Монтаж_билеты_*.xlsx
- 2Монтаж_билеты_*.xlsx
- СК_билеты_*.xlsx

### Генерируемые отчеты:
- **ОБЩИЙ_СТАЖ.xlsx** — формируется автоматически из базы
- **Реестр_по_затратам_на_билеты.xlsx** — формируется из реестров

## 🔧 Производительность

Сервер оптимизирован для работы без Docker:
- 2 worker процесса
- asyncio loop
- SQLite WAL режим
- Кэширование запросов

## 🌐 Доступ из локальной сети

Сервер запускается на `0.0.0.0:8000`, что позволяет подключаться с других устройств в сети:
```
http://<IP_сервера>:8000
```

Для определения IP сервера:
```bash
hostname -I
```

## 📝 Требования

- Python 3.9+
- pip пакеты из requirements.txt
- Нет необходимости в root правах
